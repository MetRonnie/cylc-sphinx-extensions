hieroglyph_patchÂ¶

Monkey patch hieroglyph to work with Sphinx 1.8.0+.

This extension serves as a temporary workaround, for more information see Hieroglyph issue.

If an extension provides its own visit or depart methods, patch them into the HTMLTranslator class below.

It would be possible to patch the add_node method of the Sphinx application to patch extensions automatically but in the interests of keeping the hack to a minimum this hard-coded extension should suffice.

Note

extension is automatically loaded when added to extensions, no directives or configurations required.
